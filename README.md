# Security Policies & Starter Kit (Policies • Standards • Templates)

Starter kit enxuto de **Políticas, Standards e Templates** de Segurança da Informação para implementação corporativa.
Foco: **clareza, governança de exceções (waivers), evidências e adoção prática**.

---

## 🎯 Objetivo

Ajudar a estruturar rapidamente um baseline de Segurança com documentos que:
- sejam curtos e aplicáveis (policy)
- expliquem como executar (standard/procedure)
- gerem evidências e decisões rastreáveis (templates/evidence pack)

---
